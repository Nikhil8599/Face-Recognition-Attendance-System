## Technologies Used

### **Backend:**
- **Python Flask** - Web framework
- **SQLite** - Lightweight database
- **Face Recognition** (dlib) - Face detection and recognition
- **OpenCV** - Computer vision and image processing
- **NumPy** - Numerical computing
- **Werkzeug** - WSGI utilities

### **Frontend:**
- **HTML5** - Markup language
- **CSS3** - Styling with custom glassmorphism design
- **JavaScript (ES6+)** - Client-side interactivity
- **Bootstrap 5** - Responsive CSS framework
- **Font Awesome 6** - Icons and UI elements
- **Canvas API** - Image capture and processing

### **APIs & Libraries:**
- **MediaDevices API** - Camera access
- **Fetch API** - Asynchronous HTTP requests
- **FileReader API** - File handling
- **Blob API** - Binary data handling
- **WebRTC** - Real-time camera streaming
